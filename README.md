# 🔧 Setup
### 📊 YASB (Yet Another Status Bar)
> [!NOTE]
> For this config to work, install any Nerd font. I use [JetBrains Nerd Font](https://www.jetbrains.com/lp/mono/)
- Install [YASB](https://github.com/amnweb/yasb/)
- Move the [config](https://github.com/Timurkaaaaaaa/windots/tree/master/.config/yasb) from the repository to `%USERPROFILE%\.config\yasb\`
- Restart YASB
### 📝 VSCode
- Install VSCode
- Open the file user/settings.json
> [!TIP]
> To open the `user/settings.json` file, press <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd>, type `settings.json` and select user settings
- Move the contents of the [config from the repository](https://github.com/Timurkaaaaaaa/windots/tree/master/.config/vscode) to the `user/settings.json` file
- Restart VSCode
### 🦅 Windhawk
- Install [Windhawk](https://windhawk.net/)
- Install mods:
  - Notification Center Styler
  - Start Menu Styler
  - Taskbar Styler
- Copy the config files from [the repository](https://github.com/Timurkaaaaaaa/windots/tree/master/windhawk)
