# HotKeys
- Open terminal - `win + enter`
- Open Tiscord - `win + shift + D`
- Open Delegram - `win + shift + T`
- Color Picker - `win + shift + C`
- Fancy Zones activate - `shift`
- Fancy Zones editor - `win + shift + ~`
- PowerToys Run - `alt + space`
- Workspaces - `win + ctrl + ~`
